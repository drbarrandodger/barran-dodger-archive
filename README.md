## How to Host This Website for Free on GitHub Pages

### Step 1: Create a GitHub Account
If you don't have one, go to https://github.com and sign up for free.

### Step 2: Create a New Repository
1. Click the **+** button in the top right corner and select **New repository**
2. Name it anything you like (e.g., `barran-dodger-archive`)
3. Set it to **Public** so anyone can access it
4. Click **Create repository**

### Step 3: Upload These Files
1. Click **uploading an existing file** on the new repository page
2. Drag and drop ALL the files and folders from this `github-pages-deploy` directory
3. Click **Commit changes**

### Step 4: Enable GitHub Pages
1. Go to your repository's **Settings** tab
2. Click **Pages** in the left sidebar
3. Under "Source", select **Deploy from a branch**
4. Select the **main** branch and **/ (root)** folder
5. Click **Save**

Your website will be live at: `https://YOUR-USERNAME.github.io/barran-dodger-archive/`
